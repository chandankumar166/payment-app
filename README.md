## Steps to setup the project locally

1. Clone the repo
 ```sh
 git clone https://github.com/chandankumar166/payment-app.git
 ```
3. Go to backend folder and execute the below commands
```sh
 npm i
```
```sh
node index
```
4. Go to frontend folder and execute the below commands
 ```sh
npm i
```
```sh
npm start
```
